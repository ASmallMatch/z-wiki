# ADR-0023: 书皮改程序化封面画(中性载体 + 构图原型)

- 状态:accepted
- 日期:2026-07-21
- 范围:layer2(web)首页 3D 书架的书皮纹理;supersedes ADR-0020 D1(书皮底色 = accent 混底满铺),ADR-0020 D2(6 色 accent 色板)保留
- 关联:ADR-0020(被部分 supersede)、ADR-0022 D3(昼夜呼应,沿用)、ADR-0006(纸边材质正色,不动)、CONTEXT.md「主题」节

## 背景

ADR-0020 的满铺彩色书皮(`coverBase = mixColor(accent, 底色, ~60%)`)解决了「一排黑盒子」,但引入新问题,用户对照绝区零录像带架截图提出:要么太亮太突兀,要么颜色和主题不搭,违和感严重。

对照截图量化取色的诊断:ZZZ 的录像带架 UI 骨架(架体/标签/说明字)是纯中性灰阶(`#111`~`#202021` / `#fbfbfb` / `#5d5d5d`),零色相;满饱和色(品红 `#e1055a`、青碧 `#0bada1`、暖橙 `#e3986f`)只出现在录像带封面画里,且每盘带的封面构图各不相同。塑料感的根因不是色相选择(ZZZ 封面色相族与我们色板高度重合),而是「公式混色的满铺纯色面」——彩色没有内容,天然像塑料。

方向经用户逐条确认(grilling):书皮 = 中性载体 + 程序化封面画,彩色只活在图案里。

## 决策

### D1: 书皮底色改中性载体

书皮底色不再由 accent 派生,改主题自适应中性色:Archive 深灰(近展台、与书皮图案分层)/ Draft 纸白。载体零色相、每本书相同;书与书的区分完全交给封面画。载体不固定为深色——Draft 浅色模式下深色载体不成立(用户明示约束)。

### D2: 封面画按多种构图原型生成

每本书的封面画是程序化生成的抽象构图。**3-4 种构图原型**(几何色块构成 / 斜切色带 / 渐变大字卷号 / 散点贴纸风),按书名 hash 选原型 + 定参数——一排书每本构图不同,对齐 ZZZ「每盘带都不一样」的丰富感;单一原型整排重复会成为新的塑料感(统一模板感)。

### D3: 色板不动,用法改

6 色 accent 色板(ADR-0020 D2)保留,仍按书名 hash 分配。变化在用法:图案里用更高饱和的原色直接画,不再与底色按比例混合稀释。ZZZ 封面色相族(品红/青碧/暖橙)与本色板(茜红/青碧/陶土)高度重合,违和感源自底色混色而非色相。

### D4: 标题安静区与装饰元素分级

- 每种构图原型预留一块低对比**标题安静区**(暗带/纸带),书脊标题排版维持现状压在上面——程序化构图 + 中文长标题走 ZZZ 式大字排版不可控,可读性优先。
- **保留**:顶/底纸边(ADR-0006 材质正色)、卷号标(小件锋利语言)——身份元素不动。
- **并入原型**:胶带降为构图原型的可选部件,随 hash 出现;边框、底部档案条在样张确认版中移除(装饰收敛进构图原型本身)。
- **叠加噪点做旧**:封面画上叠细噪点/印刷颗粒,杀数字纯色感(ZZZ 封面均有印刷质感)。

### D5: Draft 昼夜呼应沿用,图案粉彩化

同一本书两主题同构图、同色相(ADR-0022 D3 沿用);Draft 下图案色取柔粉彩明度映射,载体为纸白。浅底满饱和图案会变成贴纸感——「太突兀」会在 Draft 复现,故 Draft 不共用 Archive 的饱和度。

### D6: 落地先出静态样张

实现路径:先用独立脚本按同一套生成逻辑输出一排书皮的静态样张(Archive/Draft 各一版),确认视觉效果后再改 `BookShelf3D.tsx` 的 texture 生成。避免每轮调参都开 app 转书架。

## 后果

- **ADR-0020 D1 被 supersede**:`coverBase()` 满铺混色废止;D2 色板、D3 细节色自适应原则保留(具体派生对象从「混色书皮」变为「封面画图案」)。
- **「彩色书墙」语义变化**:一排书的彩色来自构图各异的封面画,底色统一中性——从「彩色书墙」变为「暗架彩画」。
- **展台色不动**:Archive `#12121a` / Draft `#ebe9e3`,中性载体在两展台上都需验证可立住(Archive 深灰载体 vs 深展台的分层是主要风险点)。
- **页面 token 零改动**:tokens.css 两主题调色板不动,只动书架 canvas 绘制。
- **后人疑问预案**:为何放弃满铺彩皮?——公式混色的纯色面无内容,天然塑料感;ZZZ 参照显示彩色必须活在「画面」里(本文背景)。为何保留旧色板?——违和感源自底色混色非色相,色板色相族与 ZZZ 封面重合(本文 D3)。为何标题不做大字排版?——中文长标题长度不可控,程序化大字风险高,安静区保可读性(本文 D4)。
