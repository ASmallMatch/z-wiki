# ADR-0022: Draft 主题改「净纸白 + 明快墨蓝」清爽现代化

- 状态:accepted
- 日期:2026-07-18
- 范围:layer2(web)Draft 主题配色从「档案室(泛黄纸 + 蓝黑墨水)」改「净纸白 + 明快墨蓝」,书架两主题色板同一色相族化;supersedes ADR-0013 的 D1''(蓝黑墨水 accent)与 D3''(泛黄纸表面 + 案卷色板)、ADR-0020 的 D2 具体色值(Archive 色板,派生机制不动);0013 的 D4''(去辉光/扫描转墨迹)、D5''(标题衬线)、D6''(纸纹提强)继续;0005 的 D2/D4/D5、0006 的 D2' 继续。
- 关联:ADR-0013(被取代 D1''/D3'')、ADR-0005、ADR-0006、ADR-0020(被取代 D2 色值)、CONTEXT.md「主题」节

## 背景

ADR-0013(2026-07-13)把 Draft 从「暖纸陶土」改「档案室(泛黄纸 #f7f3ec + 蓝黑墨水 #2b4a6f)」,修掉了陶土橙的烟熏与浑浊。落地后实际观感仍偏「尘」,四个具体短板:

1. **代码高亮糊**:`--code-kw #5a4a8a`/`--code-str #2f7a4a`/`--code-fn #2a5a8a`/`--code-op #6a4a7a` 全是低饱和中暗色,明度挤在窄带,beige 底上糊成一团。
2. **wikilink 隐身**:`a.wl` 用 `color: inherit` + `1px dotted var(--border)`(`#d9d2c0`),纸底上几乎不可见,链接不可发现。
3. **书本配色与主题不搭**:两套书架色板各自取色、色相散乱——Archive 两个蓝无绿、混黑后色相半灭;`DRAFT_ACCENTS` 混 58% 纸白后成糖果粉彩(亮薄荷/三文鱼粉),与墨蓝净纸主题脱节。
4. **整页黄色偏重**:表面层级全靠米黄明度差(`#f7f3ec`/`#fdfbf5`/`#efe9da`/`#ece4d2`),配灰蓝 accent 整体发闷。

用户定方向:保留浅色阅读定位,但跳出 0013 的做旧路线,改清爽现代--中性暖白纸 + 明快墨蓝。

## 决策

### D1:Draft accent 改明快墨蓝 `#2b6cb0`(supersedes ADR-0013 D1'')

- **Archive** accent 仍是靛青 `#6b8fc7`(不动)。
- **Draft** accent 改明快墨蓝 `#2b6cb0`,配套 `--accent-soft:#3d6fa8`(中蓝,kicker 等小字)/ `--accent-bright:#1f5aa6`(深浓,正文级 accent 文字)。
- **与 Archive 同源蓝系**:`#6b8fc7` ↔ `#2b6cb0`,切换主题 accent 不跳变(沿 0013 D1'' 的净改进)。
- 浅底语义不变:soft 浅于 accent、bright 深于 accent(浅底下「bright」取深浓取对比,同 0013 落地语义)。
- `--selection` 从 `rgba(43,74,111,0.3)` 减到 `rgba(43,108,176,0.18)`,净纸底上不再一坨深墨。

### D2:表面去黄,中性暖白(supersedes ADR-0013 D3'' 的表面部分)

| token | 0013(泛黄纸) | 0022(净纸白) |
|---|---|---|
| `--surface` | `#f7f3ec` | `#f8f7f4` |
| `--surface-alt` | `#fdfbf5` | `#ffffff`(纯白卡片提一档) |
| `--surface-hover` | `#efe9da` | `#efeeea` |
| `--surface-code` | `#efe9da` | `#f2f4f5`(一线冷灰,与暖纸底微分) |
| `--surface-shelf` | `#ece4d2` | `#ebe9e3` |
| `--border` / `--rule` | `#d9d2c0` | `#e1ded8` |
| `--text-faint` | `#78716c` | `#6f6a64`(压深补净纸底对比) |

`--text`/`--text-muted` 沿用;阴影沿用 0013 的中性偏冷(不偏褐避免烟熏)。分层原则不变(页面 < 展台 < 书皮,纸边最浅跳出,沿 ADR-0006 D3')。

### D3:书架两主题色板同一色相族化(supersedes ADR-0013 D3'' 的色板部分与 ADR-0020 D2 的具体色值;派生机制不动)

书皮与主题「不搭」的根因是两套色板各自取色、色相散乱(Archive 两个蓝无绿,Draft 糖果粉彩)。改为**同一色相族贯穿两主题**——靛蓝/青碧/松绿/陶土/茜红/堇紫,Archive 取深宝石调(混黑 62% 后色相存活),Draft 取同族柔粉彩(混纸白 58% 后协调不糖果),同一本书在两主题下同色相昼夜呼应:

| 色相 | Archive(深宝石) | Draft(柔粉彩) |
|---|---|---|
| 靛蓝(主) | `#6b8fc7`(主 accent,不动) | `#2b6cb0`(主,对齐 D1 accent) |
| 青碧 | `#3fae9d` | `#3f8f85` |
| 松绿 | `#4da568` | `#4a8f5f` |
| 陶土 | `#d09245` | `#b88a4a`(ADR-0006 材质正色传承,不动) |
| 茜红 | `#c25a68` | `#b0515a`(偶发暖) |
| 堇紫 | `#8f6fd0` | `#7a63b0` |

Archive 侧混黑 62% 后书皮呈石板蓝/深青/森林绿/皮棕/酒红/深紫(经典图书馆装帧);Draft 侧混白 58% 后呈雾蓝/青瓷/鼠尾草/沙金/灰玫/藕紫。`coverBase` 派生比例、hashAccent 分配、rim 点光(Archive `#6b8fc7` / Draft `#2b6cb0`)、纸边均不动。陶土橙传承位不动。

### D4:代码高亮重排色相

净纸底下按「色相隔开 + 明度拉开」重排,修掉 0013 的同明度低饱和糊团:

```
--code-kw:  #6d28d9  紫
--code-str: #1a7f37  绿
--code-num: #b45309  琥珀
--code-fn:  #1d4ed8  蓝
--code-cmt: #6b7280  灰
--code-op:  #0e7490  青
```

### D5:wikilink accent 着色可辨

`a.wl` 从 `color: inherit` + `dotted var(--border)` 改 `color: var(--accent)` + `dotted var(--accent-border)`,hover 加深到 `accent-bright`。保留虚线下划的克制(不实线下划、不加底色),只修「不可发现」。「不打断阅读流」的意图由虚线 + 同色 accent 维持,不再靠隐身达成。

## 后果

- **ADR-0013 D1''/D3'' 被 supersede**;0013 的 D4''(去辉光/扫描转墨迹)、D5''(标题衬线)、D6''(纸纹提强)继续 accepted。0005 的 D2/D4/D5、0006 的 D2'(换皮机制)继续。
- **浅深 accent 同源不跳变**:`#6b8fc7` ↔ `#2b6cb0` 同蓝系,切主题平滑(同 0013 D1'' 精神)。
- **陶土橙 `#b88a4a` 传承位不动**:仍居 `DRAFT_ACCENTS` 一席(ADR-0006 材质正色传承)。
- **深色 Archive 页面零改动**:`:root` 块、动画发光全保留;Archive 侧仅 `ARCHIVE_ACCENTS` 换同族深宝石色(D3),书皮仍是 accent 混黑 62% 派生(ADR-0020 D1 机制不动),页面 accent 靛青不变,是硬性验收点。wikilink 的 D5 改动两主题共享一条规则,Archive 下链接从继承色变靛青,同为可发现性净改进。
- **后人疑问预案**:为何放弃泛黄纸?--0013 落地观感偏「尘」(背景四条),用户定调清爽现代。为何代码块带一线冷灰?--与暖纸底微分,标记「非纸」区域。为何 wikilink 上色?--虚线 accent 已足够克制,隐身不是克制是不可发现(本文 D5)。为何两主题书架同一色相族?--同一座图书馆的昼夜两幅,同一本书两主题下同色相(本文 D3)。
